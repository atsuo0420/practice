# 001-ai-daily-logger

## 目的
自由入力の日報テキストを、内容に応じて「今日やったこと / 学んだこと / 困ったこと / 次回やること」に自動分類し、Notionに貼り付けやすいMarkdown形式に整形するツール。

## 使い方
1. `index.html` をブラウザで開く
2. 日付を選び、今日やったことを自由な文章で入力
3. 「整理する」を押すと、下の欄にNotion用Markdownが生成される
4. 「コピー」でクリップボードにコピーしてNotionに貼り付け

## 仕組み
- 入力文を改行・句点で1文ずつに分割
- 各文に含まれるキーワード（「学んだ」「困った」「次回」など）でカテゴリを判定
- カテゴリごとにグルーピングしてMarkdownの見出し＋箇条書きに整形
